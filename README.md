Analyzed 97,498 IT support tickets using Excel to identify patterns in workflow, agent efficiency, and user satisfaction.
Used Pivot Tables, formulas (VLOOKUP, DATEDIF, COUNTBLANK) charts, and conditional formatting to extract insights and automate analysis.
Found that System and Login Access tickets made up  70% of total volume and the average resolution time was 5 days, with hardware issues taking the longest.
Built a visual dashboard highlighting ticket trends, workload distribution, severity impact, and customer satisfaction scores.
Recommended upgrading support systems and applying targeted agent training to improve resolution speed and service consistency.
